# PyStereoVisionToolkit

Implement a stereovision acquisition and processing library in Python for research purpose.
Use the AVT PvAPI or Vimba SDK to access the Manta GigabitEthernet cameras.

Requirements :

- `NumPy`, `OpenCV`
